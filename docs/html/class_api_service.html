Detailed Description

Класс ApiService реализует API-клиент для общения с сервером.

Поддерживает авторизацию, регистрацию, отправку сообщений, получение истории чата и списка онлайн-пользователей. Работает как синглтон, чтобы обеспечить единый доступ ко всем сетевым операциям из разных частей UI.
